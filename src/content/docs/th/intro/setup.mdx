การสร้างโปรเจกต์ Astro ใหม่ใช้ `npm create astro@latest` ซึ่งเทียบเท่ากับ `npx create-next-app` หรือ `npx create-react-app` แบบเก่าของ Astro มันเปิด wizard แบบ interactive และสร้างโครงสร้างโปรเจกต์ให้

คำสั่ง `npm create astro@latest` ดึง Astro scaffolding CLI ล่าสุด (`create-astro`) จาก npm — ไม่ต้องติดตั้ง global wizard เป็นแบบสนทนาและถามเฉพาะสิ่งที่จำเป็น

## Dev server

เมื่อสร้างโครงสร้างแล้ว:

```bash
npm run dev
# or
npx astro dev
```

Dev server ของ Astro เริ่มที่ `http://localhost:4321` ตามค่าเริ่มต้น (Next.js ใช้ `:3000`) มันใช้ **Vite** เบื้องหลัง — Vite ตัวเดียวกับที่คุณอาจรู้จักจากโปรเจกต์ Vite + React — ดังนั้น hot module replacement (HMR) และการเริ่มต้นที่รวดเร็วจึงมีติดตั้งมาแล้ว

คำสั่ง dev ที่มีประโยชน์:

```bash
npx astro dev          # start the dev server
npx astro build        # build for production (outputs to dist/)
npx astro preview      # preview the production build locally
npx astro check        # TypeScript type-check all .astro files
npx astro add react    # add the React integration (adds @astrojs/react)
npx astro add tailwind # add Tailwind CSS
```

## การเพิ่ม integration

ใน Next.js การเพิ่ม React เป็นเรื่องโดยปริยาย — มันคือ framework อยู่แล้ว ใน Astro, React เป็น **integration** ที่คุณต้อง opt-in เข้ามา:

```bash
npx astro add react
```

คำสั่งนี้ติดตั้ง `@astrojs/react` และ `react`/`react-dom` และอัปเดต `astro.config.mjs` โดยอัตโนมัติ:

```js
// astro.config.mjs (after npx astro add react)
import { defineConfig } from 'astro/config';
import react from '@astrojs/react';

export default defineConfig({
  integrations: [react()],
});
```

หลังจากนี้คุณสามารถ import `.tsx` / `.jsx` React component ภายในไฟล์ `.astro` และใช้ directive `client:*` เพื่อ hydrate ได้

## TypeScript

Astro สร้าง `tsconfig.json` ที่ extends `astro/tsconfigs/strict` ไฟล์ `.astro` ของคุณถูก type-check ด้วย `npx astro check` — คำสั่งเดียวกับที่คุณรันใน CI ไม่ต้องตั้งค่า `tsc` แยกต่างหาก

<Diff title="CRA / create-next-app vs npm create astro">
- CRA: `npx create-react-app my-app` — deprecated; สร้าง SPA แบบ client-only
- Next.js: `npx create-next-app@latest` — wizard สร้าง full-stack React framework
- Astro: `npm create astro@latest` — wizard สร้าง MPA ที่เน้นเนื้อหา; React เป็น opt-in integration
- Dev server: Next ใช้ค่าเริ่มต้น `:3000`; Astro ใช้ `:4321` ทั้งคู่ใช้ Vite (Astro ตลอด; Next จาก v15+)
- Build output: Next ออก Node.js server (หรือ static export); Astro ออก `dist/` HTML แบบ static ตามค่าเริ่มต้น
</Diff>
